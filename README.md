# 🤖 Technologies

- React.js
- TypeScript
- React Redux
- TailwindCSS

# 🚀 Features

- Create new tasks
- Remove tasks
- Move tasks into the To do, In progress and Done columns

# 🛠 How to Run

```bash
# Clone this repository

$ git clone https://github.com/cassiocappellari/real-time-kanban

# Enter on the project folder

$ cd /real-time-kanban

# Install the dependencies

$ npm install

# Start the project

$ npm start

# Access the app

http://localhost:3000
```
